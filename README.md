# Event Management Dashboard

Event management dashboard built with Next.js App Router, React Query, React Hook Form, and Tailwind CSS.

## MockAPI Prerequisite

Before running the seed script, create a resource schema in `mockapi.io` called events with the following fields:

| Field | Type | Example/Faker |
| --- | --- | --- |
| `id` | Object ID | - |
| `createdAt` | Faker.js | `date.recent` |
| `eventCode` | String | - |
| `title` | String | - |
| `status` | String | - |
| `startAt` | Date | - |
| `locationText` | String | - |
| `isVirtual` | Boolean | - |
| `description` | String | - |

## Scripts

- `npm run dev`: run the local development server.
- `npm run build`: create a production build.
- `npm run start`: serve the production build.
- `npm run lint`: run ESLint.
- `npm run seed`: seed test data into MockAPI.

## Seed Data

Set the MockAPI base URL and run:

```bash
MOCKAPI_BASE_URL="https://<your-project>.mockapi.io/api/v1" npm run seed
```
